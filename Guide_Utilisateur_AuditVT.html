Présentation de la solution
Qu'est-ce qu'AuditVT Comparator ?

AuditVT Comparator est une application web autonome développée pour OT Énergie. Elle automatise la revue croisée entre un Audit Énergétique (document maître) et une Visite Technique (document à comparer), en identifiant les conformités, les écarts et les champs manquants grâce à l'intelligence artificielle.

La solution fonctionne entièrement dans le navigateur — aucune installation, aucun serveur requis. Les fichiers PDF sont analysés localement. Seul l'appel à l'API IA (OpenRouter) passe par internet.

Démarrage rapide
Lancer sa première analyse en 5 étapes
1
Ouvrir l'application
Accéder à revue-audit-energetique.vercel.app/AuditVT_Comparator.html depuis Chrome, Edge ou Firefox. L'application s'ouvre directement sur l'onglet Comparaison.
2
Configurer la clé API IA
Aller dans l'onglet Paramètres, coller la clé API (OpenRouter : sk-or-… ou OpenAI : sk-…). Le fournisseur est détecté automatiquement. Sauvegarder.
3
Charger le référentiel
Dans Paramètres, importer le fichier Excel de référentiel OT Énergie (colonnes A=Section, B=Champ, D=Check OUI/NON, F-H=Lexique VT). 140 champs seront chargés.
4
Déposer les deux PDFs
Dans l'onglet Comparaison, glisser-déposer l'Audit Énergétique (zone MAÎTRE, à gauche) et la Visite Technique (zone À COMPARER, à droite).
5
Lancer l'analyse
Cliquer sur Lancer l'analyse. L'IA traite les champs par lots. Résultats disponibles dans Rapport en quelques minutes. Exporter en Excel depuis le bouton Télécharger Excel.
⚠️
Connexion internet requise uniquement pour l'appel IA et le chargement des polices. La lecture des PDFs et l'affichage des résultats fonctionnent hors-ligne une fois la page chargée.

Onglet Comparaison
Point d'entrée principal — chargement et lancement
📂
Comparaison
Onglet principal — chargement des documents PDF
Cet onglet est le point de départ de chaque analyse. Il vous permet de charger les deux documents PDF, de vérifier que tous les prérequis sont remplis, puis de déclencher l'analyse IA.

Zone de dépôt — MAÎTRE (Audit Énergétique)

La zone de gauche reçoit le document de référence : l'Audit Énergétique complet. C'est le document qui fait foi. L'IA extrait les valeurs de ce document pour les comparer.

Zone de dépôt — À COMPARER (Visite Technique)

La zone de droite reçoit la Visite Technique à contrôler. L'IA vérifie si les données de ce document correspondent à celles de l'Audit Énergétique.

Vérification des prérequis

Avant de lancer l'analyse, l'application vérifie automatiquement 4 conditions :

PrérequisIndicateurDescription
Audit PDF🟢 vertFichier PDF chargé et texte extrait
Visite Technique PDF🟢 vertFichier PDF chargé et texte extrait
Référentiel🟢 vertAu moins 1 champ avec Check = OUI
Configuration IA🟢 vertClé API renseignée et fournisseur détecté

Onglet Rapport Détaillé
Visualisation complète des résultats champ par champ
📋
Rapport Détaillé
Analyse champ par champ avec filtres et export
Affiche l'ensemble des résultats de l'analyse, organisés par sections du référentiel puis groupés par statut. Chaque ligne indique la valeur dans l'Audit, la valeur dans la Visite Technique, le statut et le commentaire IA.

Indicateurs KPI

En haut du rapport, 4 compteurs synthétisent les résultats :

KPISignification
✅ ConformesChamps identiques ou équivalents entre Audit et VT
⚠ ApproximatifsChamps présents dans les deux mais avec des écarts mineurs
⛔ Non conformesChamps présents mais avec des valeurs significativement différentes
□ ManquantsChamps présents dans l'Audit mais absents de la Visite Technique

Score de conformité

Le score global (en %) représente la proportion de champs conformes. Trois niveaux d'appréciation :

Dossier non conforme À vérifier Recevable
0%80%95%100%

Filtres par statut

Des boutons de filtre permettent d'afficher uniquement les champs d'un statut donné : Tous · VERT · ORANGE · ROUGE · MANQUANT.

Onglet Champs Manquants
Champs présents dans l'Audit mais absents de la Visite Technique
🔍
Champs Manquants
Vue dédiée aux omissions de la Visite Technique
Liste exhaustive des champs du référentiel qui apparaissent dans l'Audit Énergétique mais sont totalement absents de la Visite Technique. Chaque entrée inclut un commentaire IA suggérant où retrouver l'information.

À quoi sert cet onglet ?

Il permet à l'opérateur de savoir ce qui manque dans la Visite Technique pour compléter le dossier. Le commentaire IA oriente sur la section ou la rubrique où l'information devrait normalement apparaître.

💡
Usage pratique : Communiquer cet onglet au technicien en charge de la Visite Technique pour qu'il complète les rubriques manquantes avant soumission finale du dossier.

Historique
50 dernières analyses sauvegardées automatiquement
Chaque analyse est automatiquement sauvegardée dans l'historique du navigateur (localStorage). L'historique conserve jusqu'à 50 entrées, avec date, noms de fichiers, score global et fichier Excel exporté.

Actions disponibles par entrée

ActionDescription
📥 TéléchargerRe-télécharger le fichier Excel exporté lors de cette analyse
🗑 SupprimerSupprimer définitivement cette entrée de l'historique
⚠️
Important : L'historique est stocké dans le navigateur (localStorage). Il sera perdu si vous videz le cache ou changez de navigateur. Pour archiver durablement, toujours exporter le rapport Excel.

Code couleur & signification
Le système de 4 statuts utilisé dans toute l'application

L'ensemble de l'application repose sur un code couleur à 4 niveaux cohérent entre le rapport HTML, le tableau de bord et le fichier Excel exporté.

✔ VERT
Conforme

Le champ est présent dans la Visite Technique et sa valeur correspond — exactement ou de façon équivalente — à celle de l'Audit Énergétique. Aucune action requise.

⚠ ORANGE
Approximatif

Le champ est présent dans les deux documents mais les valeurs présentent un écart mineur (arrondi, libellé légèrement différent, unité différente). Une vérification manuelle est recommandée.

⛔ ROUGE
Non conforme

Le champ est présent dans les deux documents mais les valeurs sont significativement différentes. Une correction est nécessaire avant soumission du dossier.

□ MANQUANT
Absent de la VT

Le champ est identifié dans l'Audit Énergétique mais est totalement absent de la Visite Technique. Il doit être ajouté dans la VT pour que le dossier soit complet.

Ordre de priorité dans le rapport

Les résultats sont présentés dans l'ordre de priorité suivant, permettant de traiter en premier les points positifs puis les anomalies :

PrioritéStatutAction recommandée
1✔ VERT — ConformesValidation — aucune action
2⛔ ROUGE — Non conformesCorrection urgente requise
3⚠ ORANGE — ApproximatifsVérification manuelle recommandée
4□ MANQUANTCompléter la Visite Technique
9
Export Excel
Fichier .xlsx complet avec 3 onglets colorés

L'export Excel génère un fichier .xlsx prêt à partager, avec mise en forme professionnelle aux couleurs OT Énergie. Le fichier est nommé automatiquement : OTEnergie_Rapport_AuditVT_YYYY-MM-DD.xlsx

Onglet 1 — Rapport de conformité

Liste complète de tous les champs analysés, groupés par statut (VERT → ROUGE → ORANGE → MANQUANT) puis par section. Chaque ligne indique le libellé, la valeur Audit, la valeur VT, le statut coloré et le commentaire IA.

Onglet 2 — Champs manquants

Liste uniquement les champs avec statut MANQUANT, avec le commentaire IA suggérant où les trouver dans la Visite Technique. Idéal pour transmission au technicien VT.

Onglet 3 — Synthèse

Tableau récapitulatif avec les KPIs principaux (totaux par statut, pourcentages), le score global et le verdict de conformité du dossier.

💡
Bouton Télécharger Excel disponible dans l'onglet Rapport. Le fichier peut être re-téléchargé à tout moment depuis l'Historique.
